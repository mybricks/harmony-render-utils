## 📋 介绍与说明

本工具包专为 **MyBricks.ai 鸿蒙生态相关产物** 提供底层运行时支持。

## 📦 安装依赖

- [@mybricks/render-utils](https://ohpm.openharmony.cn/#/cn/detail/@mybricks%2Frender-utils)

```bash
# 安装运行时依赖包
ohpm install @mybricks/render-utils
```
